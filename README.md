A top-down 2D zombie shooter built with Python and Pygame.

## Features
- **4 Difficulty Levels**: Easy, Medium, Hard, Extreme Hard
- **3 Weapon Types**: Pistol, Shotgun, Rifle
- **Wave-based combat** with increasing zombie hordes
- **Multiple zombie types**: Normal, Fast, Tank

## Setup

```bash
# Create virtual environment
python -m venv venv

# Activate (Windows)
.\venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```

## Run the Game

```bash
python src/main.py
```

## Controls
| Key | Action |
|-----|--------|
| W/A/S/D | Move |
| Mouse | Aim |
| Left Click | Shoot |
| 1/2/3 | Switch Weapon (Pistol/Shotgun/Rifle) |
| R | Reload |
| ESC | Pause / Back to Menu |
